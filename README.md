# E-Commerce Application

A full-stack e-commerce application built with Spring Boot and React.js.

## Project Overview

This is a modern e-commerce platform that provides a seamless shopping experience with features like user authentication, product management, shopping cart functionality, and order processing.

## Tech Stack

### Backend
- Spring Boot
- Spring Security
- Spring Data JPA
- MySQL
- OpenAPI (Swagger)

### Frontend
- React.js
- Vite
- Axios
- Context API for state management
- CSS for styling

## Features

- User Authentication (Login/Register)
- Product Management (CRUD operations)
- Shopping Cart Functionality
- Order Processing
- User Profile Management
- Admin Dashboard
- Responsive Design

## Project Structure

### Backend (`Ecommerce-Backend/`)
- RESTful API implementation
- Security configuration
- Database integration
- Business logic and services
- Data models and DTOs

### Frontend (`Ecommerce-Frontend/`)
- Component-based architecture
- Context-based state management
- Responsive UI components
- API integration

## Setup and Installation

### Backend
1. Navigate to `Ecommerce-Backend/`
2. Run `mvn clean install`
3. Start the application

### Frontend
1. Navigate to `Ecommerce-Frontend/`
2. Run `npm install`
3. Run `npm run dev`

## API Documentation
API documentation is available through Swagger UI when the backend is running.
